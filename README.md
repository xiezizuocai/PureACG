# PureACG

##简介：
    PureACG是一个能够提供动漫资讯、新番连载信息、高清动漫壁纸下载以及动漫歌曲排行榜的Android应用客户端。

    动漫资讯数据来源：动漫星空（每日更新）

    新番连载数据来源：Bilibili、AcFun、腾讯漫画、有妖气漫画（每日更新）

    动漫歌曲排行榜数据来源：网易云音乐ACG音乐榜（每周四更新）
